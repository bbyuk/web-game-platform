## Bounded Context 정의

| Bounded Context   | 설명                  |
|-------------------|---------------------|
| **User Context**  | 사용자 관리 (접속 및 토큰 처리) |
| **Platform Context** | 방 생성 및 입장 등 플랫폼 기능  |
| **Game Context**  | 게임 주요 기능 처리 및 기록 처리 |
| **Canvas Context** | 그림 입력 및 브로드캐스팅      |
| **Chat Context**  | 채팅 기능               |
